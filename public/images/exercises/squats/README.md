# Squats (ex_002)

## Exercise Details

**Category**: strength  
**Subcategory**: bodyweight  
**Difficulty**: beginner  
**Equipment**: none

### Target Muscles
- **Primary**: legs, glutes
- **Secondary**: core, back

### Exercise Metrics
- **Calories Burned Per Minute**: 8
- **Default Duration**: 300 seconds
- **Default Sets**: 3
- **Default Reps**: 15
- **Rest Time**: 60 seconds

### Instructions
1. Stand with feet shoulder-width apart
2. Lower body by bending knees and hips
3. Keep chest up and back straight
4. Push through heels to return to standing

### Tips
- Keep knees aligned with toes
- Don't let knees cave inward
- Go as low as comfortable

### Variations
- **Jump Squats** (Difficulty: harder)
- **Goblet Squats** (Difficulty: same)
- **Wall Sits** (Difficulty: easier)
