# Bicycle Crunches (ex_010)

## Exercise Details

**Category**: strength  
**Subcategory**: bodyweight  
**Difficulty**: beginner  
**Equipment**: none

### Target Muscles
- **Primary**: core
- **Secondary**: obliques

### Exercise Metrics
- **Calories Burned Per Minute**: 6
- **Default Duration**: 180 seconds
- **Default Sets**: 3
- **Default Reps**: 20
- **Rest Time**: 45 seconds

### Instructions
1. Lie on back with hands behind head
2. Bring one knee toward chest while rotating opposite elbow to knee
3. Alternate sides in cycling motion
4. Keep core engaged throughout

### Tips
- Don't pull on neck
- Focus on rotation
- Controlled movement

### Variations
- **Slow Bicycle Crunches** (Difficulty: harder)
