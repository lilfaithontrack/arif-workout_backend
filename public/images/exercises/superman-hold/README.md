# Superman Hold (ex_020)

## Exercise Details

**Category**: strength  
**Subcategory**: bodyweight  
**Difficulty**: beginner  
**Equipment**: none

### Target Muscles
- **Primary**: back
- **Secondary**: glutes, shoulders

### Exercise Metrics
- **Calories Burned Per Minute**: 4
- **Default Duration**: 120 seconds
- **Default Sets**: 3
- **Default Reps**: 1
- **Rest Time**: 45 seconds

### Instructions
1. Lie face down with arms extended overhead
2. Simultaneously lift arms, chest, and legs off ground
3. Hold position
4. Lower back down with control

### Tips
- Keep neck neutral
- Don't strain
- Focus on back muscles

### Variations
- **Alternating Superman** (Difficulty: easier)
- **Swimming Superman** (Difficulty: harder)
