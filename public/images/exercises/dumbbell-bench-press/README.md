# Dumbbell Bench Press (ex_007)

## Exercise Details

**Category**: strength  
**Subcategory**: weights  
**Difficulty**: intermediate  
**Equipment**: dumbbells, bench

### Target Muscles
- **Primary**: chest
- **Secondary**: triceps, shoulders

### Exercise Metrics
- **Calories Burned Per Minute**: 6
- **Default Duration**: 300 seconds
- **Default Sets**: 4
- **Default Reps**: 10
- **Rest Time**: 90 seconds

### Instructions
1. Lie on bench with dumbbells at chest level
2. Press weights up until arms extended
3. Lower weights back to chest level
4. Repeat for desired reps

### Tips
- Keep feet flat on floor
- Don't arch back excessively
- Control the weight on descent

### Variations
- **Incline Dumbbell Press** (Difficulty: same)
- **Single-arm Dumbbell Press** (Difficulty: harder)
