# Glute Bridges (ex_018)

## Exercise Details

**Category**: strength  
**Subcategory**: bodyweight  
**Difficulty**: beginner  
**Equipment**: none

### Target Muscles
- **Primary**: glutes
- **Secondary**: hamstrings, core

### Exercise Metrics
- **Calories Burned Per Minute**: 5
- **Default Duration**: 240 seconds
- **Default Sets**: 3
- **Default Reps**: 15
- **Rest Time**: 45 seconds

### Instructions
1. Lie on back with knees bent, feet flat
2. Push through heels to lift hips
3. Squeeze glutes at top
4. Lower back down with control

### Tips
- Don't arch lower back
- Squeeze glutes hard at top
- Keep core engaged

### Variations
- **Single-leg Glute Bridge** (Difficulty: harder)
- **Weighted Glute Bridge** (Difficulty: harder)
