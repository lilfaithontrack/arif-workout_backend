# Box Jumps (ex_019)

## Exercise Details

**Category**: hiit  
**Subcategory**: plyometric  
**Difficulty**: advanced  
**Equipment**: box

### Target Muscles
- **Primary**: legs, glutes
- **Secondary**: core, calves

### Exercise Metrics
- **Calories Burned Per Minute**: 11
- **Default Duration**: 240 seconds
- **Default Sets**: 3
- **Default Reps**: 10
- **Rest Time**: 90 seconds

### Instructions
1. Stand facing box at appropriate height
2. Swing arms and jump onto box
3. Land softly with both feet
4. Step down and repeat

### Tips
- Start with lower box height
- Land softly
- Full hip extension at top

### Variations
- **Step-ups** (Difficulty: easier)
- **Depth Jumps** (Difficulty: harder)
