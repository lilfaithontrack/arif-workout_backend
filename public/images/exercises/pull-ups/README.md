# Pull-ups (ex_009)

## Exercise Details

**Category**: strength  
**Subcategory**: bodyweight  
**Difficulty**: intermediate  
**Equipment**: pull_up_bar

### Target Muscles
- **Primary**: back, biceps
- **Secondary**: shoulders, core

### Exercise Metrics
- **Calories Burned Per Minute**: 7
- **Default Duration**: 240 seconds
- **Default Sets**: 3
- **Default Reps**: 8
- **Rest Time**: 90 seconds

### Instructions
1. Hang from bar with overhand grip
2. Pull body up until chin over bar
3. Lower back down with control
4. Repeat

### Tips
- Engage lats throughout movement
- Avoid swinging
- Full range of motion

### Variations
- **Assisted Pull-ups** (Difficulty: easier)
- **Weighted Pull-ups** (Difficulty: harder)
- **Chin-ups** (Difficulty: easier)
