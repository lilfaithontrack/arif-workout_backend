# Lunges (ex_004)

## Exercise Details

**Category**: strength  
**Subcategory**: bodyweight  
**Difficulty**: beginner  
**Equipment**: none

### Target Muscles
- **Primary**: legs, glutes
- **Secondary**: core

### Exercise Metrics
- **Calories Burned Per Minute**: 7
- **Default Duration**: 300 seconds
- **Default Sets**: 3
- **Default Reps**: 10
- **Rest Time**: 60 seconds

### Instructions
1. Step forward with one leg
2. Lower hips until both knees bent at 90 degrees
3. Push back to starting position
4. Alternate legs

### Tips
- Keep front knee over ankle
- Don't let knee extend past toes
- Maintain upright posture

### Variations
- **Reverse Lunges** (Difficulty: same)
- **Walking Lunges** (Difficulty: harder)
- **Jump Lunges** (Difficulty: advanced)
