# Mountain Climbers (ex_006)

## Exercise Details

**Category**: cardio  
**Subcategory**: bodyweight  
**Difficulty**: intermediate  
**Equipment**: none

### Target Muscles
- **Primary**: core, legs
- **Secondary**: shoulders, arms

### Exercise Metrics
- **Calories Burned Per Minute**: 10
- **Default Duration**: 180 seconds
- **Default Sets**: 3
- **Default Reps**: 20
- **Rest Time**: 60 seconds

### Instructions
1. Start in plank position
2. Drive one knee toward chest
3. Quickly switch legs
4. Continue alternating at a fast pace

### Tips
- Keep hips level
- Maintain plank position
- Start slow and build speed

### Variations
- **Slow Mountain Climbers** (Difficulty: easier)
- **Cross-body Mountain Climbers** (Difficulty: harder)
