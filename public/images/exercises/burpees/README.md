# Burpees (ex_005)

## Exercise Details

**Category**: hiit  
**Subcategory**: full_body  
**Difficulty**: intermediate  
**Equipment**: none

### Target Muscles
- **Primary**: full_body
- **Secondary**: core, legs

### Exercise Metrics
- **Calories Burned Per Minute**: 12
- **Default Duration**: 300 seconds
- **Default Sets**: 3
- **Default Reps**: 10
- **Rest Time**: 90 seconds

### Instructions
1. Start standing, drop into squat
2. Place hands on ground, jump feet back to plank
3. Do a push-up
4. Jump feet forward, explode up with jump

### Tips
- Maintain good form throughout
- Modify by stepping instead of jumping
- Breathe consistently

### Variations
- **Half Burpees** (Difficulty: easier)
- **Burpee Box Jumps** (Difficulty: harder)
