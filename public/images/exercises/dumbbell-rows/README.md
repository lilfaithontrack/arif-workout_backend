# Dumbbell Rows (ex_012)

## Exercise Details

**Category**: strength  
**Subcategory**: weights  
**Difficulty**: intermediate  
**Equipment**: dumbbells, bench

### Target Muscles
- **Primary**: back
- **Secondary**: biceps, core

### Exercise Metrics
- **Calories Burned Per Minute**: 6.5
- **Default Duration**: 300 seconds
- **Default Sets**: 4
- **Default Reps**: 12
- **Rest Time**: 75 seconds

### Instructions
1. Place one knee and hand on bench
2. Hold dumbbell in opposite hand
3. Pull dumbbell to hip, keeping elbow close to body
4. Lower with control and repeat

### Tips
- Keep back flat
- Pull with back, not arm
- Squeeze shoulder blade at top

### Variations
- **Bent-over Rows** (Difficulty: same)
- **Renegade Rows** (Difficulty: harder)
