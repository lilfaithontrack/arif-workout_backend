# High Knees (ex_016)

## Exercise Details

**Category**: cardio  
**Subcategory**: warm_up  
**Difficulty**: beginner  
**Equipment**: none

### Target Muscles
- **Primary**: legs
- **Secondary**: core, hip_flexors

### Exercise Metrics
- **Calories Burned Per Minute**: 9
- **Default Duration**: 180 seconds
- **Default Sets**: 3
- **Default Reps**: 30
- **Rest Time**: 45 seconds

### Instructions
1. Stand with feet hip-width apart
2. Run in place, bringing knees up to hip level
3. Pump arms in running motion
4. Maintain fast pace

### Tips
- Land on balls of feet
- Keep core tight
- Drive knees up high

### Variations
- **Slow High Knees** (Difficulty: easier)
- **High Knee Sprints** (Difficulty: harder)
