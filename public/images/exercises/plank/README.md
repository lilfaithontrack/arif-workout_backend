# Plank (ex_003)

## Exercise Details

**Category**: strength  
**Subcategory**: bodyweight  
**Difficulty**: beginner  
**Equipment**: none

### Target Muscles
- **Primary**: core
- **Secondary**: shoulders, back

### Exercise Metrics
- **Calories Burned Per Minute**: 5
- **Default Duration**: 60 seconds
- **Default Sets**: 3
- **Default Reps**: 1
- **Rest Time**: 45 seconds

### Instructions
1. Start in forearm plank position
2. Keep body in straight line from head to heels
3. Engage core and hold position
4. Breathe normally throughout

### Tips
- Don't let hips sag or pike up
- Keep neck neutral
- Focus on quality over duration

### Variations
- **Side Plank** (Difficulty: same)
- **Plank to Downward Dog** (Difficulty: harder)
