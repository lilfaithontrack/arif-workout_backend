# Shoulder Press (ex_014)

## Exercise Details

**Category**: strength  
**Subcategory**: weights  
**Difficulty**: intermediate  
**Equipment**: dumbbells

### Target Muscles
- **Primary**: shoulders
- **Secondary**: triceps, core

### Exercise Metrics
- **Calories Burned Per Minute**: 6
- **Default Duration**: 300 seconds
- **Default Sets**: 4
- **Default Reps**: 10
- **Rest Time**: 90 seconds

### Instructions
1. Stand or sit with dumbbells at shoulder height
2. Press weights overhead until arms extended
3. Lower back to shoulder height
4. Repeat

### Tips
- Don't arch back
- Keep core tight
- Full range of motion

### Variations
- **Arnold Press** (Difficulty: same)
- **Single-arm Press** (Difficulty: harder)
