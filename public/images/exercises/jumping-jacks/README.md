# Jumping Jacks (ex_011)

## Exercise Details

**Category**: cardio  
**Subcategory**: warm_up  
**Difficulty**: beginner  
**Equipment**: none

### Target Muscles
- **Primary**: full_body
- **Secondary**: legs, shoulders

### Exercise Metrics
- **Calories Burned Per Minute**: 8
- **Default Duration**: 180 seconds
- **Default Sets**: 3
- **Default Reps**: 30
- **Rest Time**: 30 seconds

### Instructions
1. Start standing with feet together, arms at sides
2. Jump feet apart while raising arms overhead
3. Jump back to starting position
4. Repeat at steady pace

### Tips
- Land softly
- Keep core engaged
- Maintain steady rhythm

### Variations
- **Step Jacks** (Difficulty: easier)
- **Star Jumps** (Difficulty: harder)
