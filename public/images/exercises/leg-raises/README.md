# Leg Raises (ex_013)

## Exercise Details

**Category**: strength  
**Subcategory**: bodyweight  
**Difficulty**: intermediate  
**Equipment**: none

### Target Muscles
- **Primary**: core
- **Secondary**: hip_flexors

### Exercise Metrics
- **Calories Burned Per Minute**: 5.5
- **Default Duration**: 180 seconds
- **Default Sets**: 3
- **Default Reps**: 12
- **Rest Time**: 60 seconds

### Instructions
1. Lie on back with legs extended
2. Keep lower back pressed to floor
3. Raise legs to 90 degrees
4. Lower slowly without touching floor

### Tips
- Don't arch lower back
- Control the descent
- Keep legs straight

### Variations
- **Bent-knee Leg Raises** (Difficulty: easier)
- **Hanging Leg Raises** (Difficulty: harder)
