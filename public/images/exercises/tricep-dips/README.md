# Tricep Dips (ex_017)

## Exercise Details

**Category**: strength  
**Subcategory**: bodyweight  
**Difficulty**: intermediate  
**Equipment**: bench

### Target Muscles
- **Primary**: triceps
- **Secondary**: chest, shoulders

### Exercise Metrics
- **Calories Burned Per Minute**: 6
- **Default Duration**: 240 seconds
- **Default Sets**: 3
- **Default Reps**: 12
- **Rest Time**: 60 seconds

### Instructions
1. Place hands on bench behind you
2. Extend legs forward, heels on ground
3. Lower body by bending elbows to 90 degrees
4. Push back up to starting position

### Tips
- Keep elbows pointing back
- Don't shrug shoulders
- Control the movement

### Variations
- **Bent-knee Dips** (Difficulty: easier)
- **Weighted Dips** (Difficulty: harder)
