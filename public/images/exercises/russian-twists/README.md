# Russian Twists (ex_015)

## Exercise Details

**Category**: strength  
**Subcategory**: bodyweight  
**Difficulty**: beginner  
**Equipment**: none

### Target Muscles
- **Primary**: core, obliques
- **Secondary**: hip_flexors

### Exercise Metrics
- **Calories Burned Per Minute**: 6.5
- **Default Duration**: 180 seconds
- **Default Sets**: 3
- **Default Reps**: 20
- **Rest Time**: 45 seconds

### Instructions
1. Sit with knees bent, feet off floor
2. Lean back slightly, keeping back straight
3. Rotate torso side to side
4. Touch floor beside hips each rotation

### Tips
- Keep core engaged
- Controlled rotation
- Don't rush

### Variations
- **Weighted Russian Twists** (Difficulty: harder)
- **Feet-on-floor Russian Twists** (Difficulty: easier)
